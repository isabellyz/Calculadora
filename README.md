# Calculadora Web

Este é um projeto simples de uma **calculadora** desenvolvida usando **HTML**, **CSS** e **JavaScript**. O objetivo desse projeto é criar uma calculadora funcional e intuitiva com uma interface simples.

## Funcionalidades

- Realizar operações matemáticas básicas: **adição**, **subtração**, **multiplicação** e **divisão**.
- Exibir o resultado da operação quando o botão "=" for pressionado.
- Limpar a tela com o botão "C".

## Tecnologias Utilizadas

- **HTML**: Estrutura da página e layout da calculadora.
- **CSS**: Estilo e design da interface.
- **JavaScript**: Lógica da calculadora para realizar as operações matemáticas e atualizar a tela.
